## Knowledge Gap Warning

The LLM's training data likely covers Godot up to ~4.3. Versions 4.4, 4.5,
and 4.6 introduced changes that may not be reliably known without these
reference docs. Always cross-reference this directory before suggesting Godot API usage.

## 4.6.2 Verification Note

Godot 4.6.2 is a maintenance release (61 contributors, 122 fixes) and is
reported as a safe upgrade from 4.6.1 with no known incompatibilities at
publication time.

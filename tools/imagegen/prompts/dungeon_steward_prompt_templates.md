# Dungeon Steward prompt templates

Use these as starting points, not scripture. The models are suggestible, not wise.

## Global style guidance

Shared positive tags:
- browser-first game art
- readable silhouette
- controlled value grouping
- restrained fantasy detail
- clean focal hierarchy
- dark neutral or parchment-friendly background when ideating isolated assets

Shared negative tags:
- text
- watermark
- signature
- border
- muddy composition
- unreadable silhouette
- low contrast focal point
- oversaturated neon unless intentionally testing magic effects

## 1) Relic concept

Template:
`fantasy relic concept art, [object], frontier dungeon stewardship theme, readable silhouette, aged brass and iron, restrained teal rune glow, elegant engraving, isolated asset presentation, dark neutral background, high clarity concept art`

Use for:
- relic ideation
- upgrade tokens
- key objects
- reward icons before pixel cleanup

## 2) Card illustration base

Template:
`fantasy card illustration, [action or spell], strong central subject, readable at small size, limited background clutter, tactical clarity, dark fantasy but not grimdark, polished digital concept art`

Use for:
- attack/skill/spell concepts
- class/archetype visual motifs
- composition thumbnails for final card art

## 3) Enemy portrait / encounter concept

Template:
`fantasy enemy concept art, [enemy idea], readable silhouette, combat gameplay readability, one clear threat motif, frontier dungeon ecosystem, muted materials with one accent color, presentation sheet style`

Use for:
- elite and boss ideation
- encounter family sheets
- telegraph readability exploration

## 4) Hub concept

Template:
`fantasy outpost hub concept art, recovering dungeon frontier, practical stewardship fantasy, warm lamplight against cool stone, readable layout, compact management space, handcrafted materials, game environment concept art`

Use for:
- meta hub moodboards
- architecture explorations
- menu backdrop ideation

## 5) UI ornament / frame motif exploration

Template:
`fantasy UI ornament concept, [motif], clean ornamental frame design, brass, ink, parchment, restrained rune accents, readable geometry, no typography, transparent-background-friendly composition`

Use for:
- card frame motifs
- panel corners
- button embellishments
- reward banner ornaments

## Quick note on pixel art

Do not ask the model for final production pixel art and expect it to behave like a civilized grid.

Preferred process:
1. generate concept or paintover base
2. select shapes and materials
3. reduce complexity manually
4. rebuild final pixel asset in Aseprite
